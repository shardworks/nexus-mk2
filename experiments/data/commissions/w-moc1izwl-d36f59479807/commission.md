Decision D4 punts `linkUrl: null` for MVP because there is no guild-wide concept of a public web base URL. The Oculus reads its own `port` from config but nothing composes `scheme://host:port` for external embeds. A future commission should add `settings.webBaseUrl` (framework-level) or `lattice.webBaseUrl` (apparatus-level) so pulse emitters can produce deep-links that work on remote channels (Discord, ntfy, email).
