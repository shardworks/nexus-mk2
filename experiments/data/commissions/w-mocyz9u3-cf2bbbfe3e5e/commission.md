`packages/plugins/astrolabe/src/engines.test.ts` is a single file covering four engines (plan-init, inventory-check, decision-review, observation-lift) and is 1562 lines long. It imports a shared test harness, shared clerk mocks, and shared helpers. The `observation-lift` block alone (lines 1237–1562) is the size of a full test file. Splitting it into `engines/plan-init.test.ts`, `engines/inventory-check.test.ts`, `engines/decision-review.test.ts`, and `engines/observation-lift.test.ts` would make it much easier to navigate, let the shared harness (`setup()`, `makePlan`, the mock clerk) live in a small helper module, and match the per-engine file layout of `src/engines/*.ts`. This commission will substantially rewrite the observation-lift block anyway, so it is a natural time to extract it.
