Brief suggests future commissions should split the writs IIFE "mirroring the structure of `packages/plugins/ratchet/pages/clicks/index.html`." That file is 1043 total lines with its entire script body inside a single `(() => { ... })()` block (lines 189–1040). It is not split into ES modules nor an external `.js` file. The clicks page and the writs page currently share an identical script-structure pattern — the "mirror" reference does not provide a precedent for how to split. The real repo precedent for external scripts lives in astrolabe/animator/spider/parlour (each of which has a sibling `.js` file loaded via `<script src="…">`). A future commission choosing to split the writs IIFE should cite those as the structural model and — if the clicks page grows past its own threshold first — may want to apply the same split there.
