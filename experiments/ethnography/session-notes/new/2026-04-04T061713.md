- **Updated all three plan agent instructions** — removed hardcoded `/workspace/nexus/` paths and Project Context section, added Working Environment section, updated all output paths to `{specs_dir}/{slug}/`.

## Decisions made and/or deferred

- **Temp clone over path rewriting** — Sean pushed for running agents inside the codex rather than rewriting paths at dispatch time. Cleaner, generalizable.
- **CLAUDE.md auto-discovery for plan agents** — plan agents don't use `--setting-sources user`, so the codex's CLAUDE.md is picked up naturally. No hardcoded doc references needed in agent instructions.
- **No orientation section in nexus CLAUDE.md** — Sean pointed out this would info-dump every agent. Anima curriculum is the right layer for prescribing reading.
- **Old specs left as-is** — all already dispatched/completed; not worth regenerating.
