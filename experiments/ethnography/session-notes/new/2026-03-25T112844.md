## Decisions made and/or deferred

- **stream-json over json output format:** Used `--output-format stream-json --verbose` rather than `--output-format json` for autonomous sessions. This captures the full conversation transcript (assistant + user messages with tool use), not just the final result text. Requires `--verbose` flag per Claude CLI requirements.
- **Duplicate session.started is not a bug:** The two events were from genuinely different sessions. No code change needed.
- **session.started/session.ended as clockwork events:** Raised but deferred — these currently have no standing orders and just consume event IDs. Could be useful for future automation but are noise for now.
