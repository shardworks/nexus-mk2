## Decisions made and/or deferred

- **Manifest is a core capability, not an engine.** It's a pure function (anima name → system prompt + tools). Absorbed into core.
- **`engine-mcp-server` absorbed into `engine-session-claude-code`.** MCP is a Claude Code transport detail, not shared infrastructure.
- **All sessions through one funnel** — `core.launchSession()` handles workspace lifecycle, Ledger recording, events, metrics, and transcript capture for every session type.
- **Workspace resolution via event payload convention** — standard optional `workshop`/`worktreePath` fields on any event. `worktreePath` = use as-is (caller owns lifecycle), `workshop` only = fresh temp worktree from main, neither = guildhall.
- **Fresh temp worktrees every session** — no persistent main worktree. Avoids staleness and conflicts.
- **Temp worktree naming uses crypto-safe hash**, not timestamps.
- **`SessionRecord`** (flat structure) written to `.nexus/sessions/` with provenance including curriculum/temperament name+version.
- **`commission_sessions` join table** links commissions to sessions instead of a `commission_id` FK on sessions.
- **Roles stored as JSON text column** — frozen snapshot, small list, not worth a join table.
- **`model` field added** to sessions table for cost analysis and experiment tracking.
