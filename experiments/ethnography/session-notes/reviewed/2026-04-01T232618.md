## Decisions made and/or deferred

- **Ditched `tools-available`** — animas discover tools via MCP native listing; no need to rebuild as a guild tool
- **`tools:read` permission** — not permissionless, not caller-gated. Off by default for animas, available to admin roles that get the grant
- **No handler context mechanism** — explored ToolContext, AsyncLocalStorage, resolution-time binding; all unnecessary once `tools-available` was killed
