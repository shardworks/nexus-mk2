## Decisions made and/or deferred

- **Workshops as `Record<string, WorkshopEntry>`** not `string[]`. Stores `remoteUrl` (source of truth for rehydration) and `addedAt`. No description field for now; default branch assumed to be `main`.
- **`nsg guild` as a new command group** for guild-wide operations. `restore` is its first subcommand.
- **`create` requires `gh` auth** — checked at runtime before attempting repo creation. Private repos by default.
- **Bare clones are ephemeral.** `guild.json` is the source of truth; `.nexus/workshops/` is reconstructible via `nsg guild restore`.
