## Decisions made

- **HandlerContext deleted.** Handlers access Stacks via `guild().apparatus<StacksApi>('stacks')`. Transaction binding is transparent via AsyncLocalStorage.
- **OR queries in v1** as apparatus-level multi-query merge (backend stays AND-only). Free perf upgrade path later.
- **Compound indexes in v1** via `(string | string[])[]` syntax.
- **Non-awaited writes = contract violation**, documented as spec note rather than test case (race conditions are timing-dependent).
- **Spec changes left unstaged** in `/workspace/nexus/` per Sean's direction for manual review.
