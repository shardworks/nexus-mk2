## What we did

- **Token usage investigation** — analyzed spending across all agent categories by mining `~/.claude` session transcripts, guild DB (`books_animator_sessions`), and per-project JSONL files. Built a breakdown: instruments ($368), plan pipeline ($267), Coco ($158), guild artificer ($41).

- **Identified cache-sharing as the real cost lever** — current architecture has zero prefix sharing across 9 calls per commission. Unified context prefix + sequential execution = 5x cost reduction without changing models.
- **Filed URGENT spec** at `.scratch/todo/URGENT-unified-instrument-context.md` — commissioning blocked until resolved.

## Decisions made and/or deferred

- **Cache prefix unification: the priority fix.** Restructure prompts so all instruments share a common context preamble, differentiated only by a rubric suffix. Run sequentially to exploit 1-hour cache TTL.
- **Deferred: root cause of Sonnet integration scorer failures.** 2/3 runs failed consistently for one commission. Likely timeout but not confirmed.
