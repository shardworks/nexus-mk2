# Naming Vibe Check

## Concepts That Need Names

| Concept | Description |
|---------|-------------|
| The system itself | The whole multi-agent factory |
| A request to the system | What Sean asks the factory to produce |
| The output/deliverable | The usable artifact that comes back |
| An agent | An autonomous worker |
| A group of agents on a task | Multiple agents collaborating |
| The human operator | Sean's role |
| The agent workspace | Where agents build things |
| The human workspace | The workshop (nexus-mk2) |
| The genie channel | Agents requesting things from Sean |
| An experiment | A structured test of the system |
| System logs/records | What happened, what it cost |

## Mix-and-Match Picks (Coco's favorites across genres)

These are terms that read cleanly regardless of genre commitment:

- **Commission** (request) — works in industrial, fantasy, and plain English
- **Artifact** (output) — already natural in software, gains flavor from fantasy
- **Fruit** (output) — already in the philosophy, organic and clear
- **Run** (experiment) — clean, universal
- **Patron** (human operator) — commission + patron is a strong pair
- **Forge** (agent workspace) — evocative but not silly
- **Ledger** (records) — grounded, implies accountability
