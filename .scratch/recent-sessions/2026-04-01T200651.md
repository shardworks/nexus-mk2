## Decisions made and/or deferred

- **Animator owns weave+animate combination** ‚Äî not a separate Summoner apparatus. It's already "the session apparatus."
- **Work prompts bypass The Loom** ‚Äî the Loom only handles identity composition (system prompt). Clean separation.
- **`AnimaWeave` naming** ‚Äî chosen over `WovenAnima` and `ManifestedAnima`. Concise, maps to the Loom metaphor.
- **No function overloads for streaming** ‚Äî uniform `{ chunks, result }` return. Overloads break down with dynamic `streaming` flag values.
- **Synchronous return** ‚Äî `animate()` and `summon()` return the handle immediately; async work lives inside the `result` promise and `chunks` iterable.
- **Deferred:** Streaming through the tool contract (documented as future in arch spec). Current CLI streaming uses stderr side-channel from the provider.
