## Decisions made and/or deferred

- **Remove framework/guild split** — no more `source: 'nexus'|'guild'`, no `nexus/` prefix dirs, no removal protection. All artifacts equal. Recovery via bundle reinstall.
- **Bundles don't need npm package.json deps** — the manifest is the single source of truth. No authoring redundancy.
- **Git URLs suffice for package refs** — no npm publishing required for implements/engines in bundles.
- **Inline limited to content types** — implements/engines cannot be inline (dep resolution problem). Schema enforces this.
- **Deferred: npm workspaces** — guild as workspace root deferred until inline-implements-with-deps becomes a real need. Layout is already compatible.
- **Deferred: advisor anima** — design complete, implementation pending bundle system.
