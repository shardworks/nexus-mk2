## Decisions made and/or deferred

- **Bare clones over fresh clones** — `.nexus/codexes/<name>.git` with worktrees branching from them.
- **Linear history only** — ff-only merge or rebase+ff; sealing seizes on conflicts. No merge commits, no force push.
- **Config shape**: `codexes: { settings: { ... }, registered: { <name>: { remoteUrl } } }` — `registered` key chosen over `catalog`/`repos`.
- **Branch optional on `openDraft`** — auto-generates `draft-<ulid>` if omitted.
- **Loose coupling between Scriptorium and Animator** — `DraftRecord.path` is the `cwd` handoff; orchestration is the caller's job.
- **`codex-push` not `draft-push`** — pushing is a codex-level operation, not a draft operation.
- **Reject branch collisions**; **external draft cleanup**; **config write API in nexus-core** (not Arbor).
- **Clockworks events are future state** — documented but not MVP.
