# Clerk page for oculus

The clerk should have a 'recommends' dependency on Oculus, and contribute a page for managing writs. This should include a display of writs and their key metadata. Sorting, filtering, and search (particularly by status) should be implemented. Ther should be a way to perform key writ operations:

- legal state transitions, especially cancel
- create/edit/delete links between writs
- repost failed writs (creates a new writ with the same body/title--with possible annotation of it being a repost--, along with an appropriate link to the original writ)
- Post new writs: this will require a large, possibly expandable text area, a dropdown to select the writ type (from the available options in the giuld), and a submit button
