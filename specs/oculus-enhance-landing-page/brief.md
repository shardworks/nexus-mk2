# Oculus: Enhance landing page

Currently, the oculus main page only has a 'pages' widget which is not very useful. Let's remove that, and replace it with an overview of the guild's configuration and status. Key elements would be things like:

- installed plugins, and their status
- startup warnings, if any
- general information similar to what the status tool returns
- general configuration summary, and possibly even a (read-only? read-write?) display of the guild.json
