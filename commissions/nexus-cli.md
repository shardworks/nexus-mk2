## What I Need

A CLI tool that reports its own name and version information when executed.

## Requirements

1. When run, the tool prints its name and version.
2. The name of the tool is "Nexus CLI"
3. Version information must include a timestamp indicating when the tool was last updated (built, published, modified — however you interpret "last updated" is fine, but it must reflect a real dynamic time, not a hardcoded string).
4. I must be able to run it with a single command from my machine.
5. The GitHub repository must include a README explaining how to run the tool, including any necessary environment setup.

## Constraints

- Build this in the repository you've been given.
- You choose the language, tooling, and delivery mechanism.

## How I'll Evaluate

- I will read the README.
- I will follow its instructions to run the tool.
- I will check that the output includes a name, version, and a meaningful timestamp.
