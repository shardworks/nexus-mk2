# Distill worked example — laboratory archive design session

This folder contains a worked example of the proposed Tier-1 distill +
verify pipeline, applied to a real session.

**Source session:** `0cb4907e-adf3-41d2-92f3-6e88dc76565b` (2026-05-01)
**Click umbrella:** `c-mom18vmh` "Build the Laboratory apparatus" — the
specific click resolved was `c-momaa5o9` "Design the archive engine."

**Files:**
- `raw-conversation.md` — backbone of the original transcript (86 messages,
  tool-call summaries elided for readability).
- `annotated-conversation.md` — same conversation, but with BuJo-style
  inline structural markers (`[memo]`, `[?]`, `[!]`, `[•]`, `[>]`) on
  Coco's lines as if she had been writing with that discipline.
- `distill.md` — the synthesized Tier-1 distill artifact, hybrid form
  (Intent / Questions / Decisions / Next steps).
- `verify.md` — what a verify subagent would produce: grounding /
  contradiction / omission findings against the source transcript.

The point is to show what the system *produces* under the proposal, not
to argue for adoption. Reading order: skim raw-conversation → read
annotated to see the symbol pattern → read distill to see what falls
out → read verify to see what it catches.
